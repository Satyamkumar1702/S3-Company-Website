Template Installation

FTP Upload:

  • Open up your FTP manager and connect to your hosting
  • Browse to required directory (Normally public_html)
  • Upload the files inside Techtrix folder.

Customization

Open / Techtrix / index.html file and set the configuration options if you'd like. There's possibility to set your personal logo, & your Copyright Text, and choose to have smooth scroll or not, etc. The index.html file has a complete documentation to help you configuring your template.

How to change Header Logo?

This template uses two logos one black and one white logo you can change the logo as per your wish and it is in the images folder. Open / Techtrix / assets / img /

How to change Youtube Video Link?

There is a YouTube random video link in the home section, if you want you can change this link by editing the index.html file

How to change Footer Copyright Text?

This template is given a random copyright name, you can change it from here index.html - All Page Footer Section

How to change Google MAP?

Open / Techtrix / contact.html / map section file edit & change your Map embeb iframe.

Set your preferred location from Google Maps and copy this location share to Embed a Map iframe. Google Map Link

How to change PHP Random Mail?

PHP mail file by editing it you can give your work mail so you need to edit phpmail.php files and change random mail and now add your mail line no 23 & done.
